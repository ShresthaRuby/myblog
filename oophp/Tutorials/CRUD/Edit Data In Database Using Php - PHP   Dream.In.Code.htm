Another debugging tip is in update_ac.php you could try dumping $sql to also make sure the query is complete. Always check variables to make sure they exists before using them. PHP functions isset or empty could be used to do this.

Always always sanitize user input. Look into Prepared Statements. They will avoid the risk of having to deal with SQL injections. They're offered by MySQLi and PDO.

Database credentials should be centralized also. Meaning they should be in one location. What if your password changed? You would then have to go into every script and change them all. Instead place them in a PHP file then include it where ever you need it.
view source
print?
1<?php
2#credentials.php
3define('DB_HOST', 'localhost');
4define('DB_USERNAME', 'root');
5//you get the point
6?>

view source
print?
1<?php
2require_once 'credentials.php';
3$mysqli = new MySQLi(DB_HOST, DB_USERNAME.......
4?>
